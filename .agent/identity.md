EJTunes Project Identity

1. 프로젝트 정체성 (Identity)
• 앱 이름: EJTunes

  • 선정 이유: 'iTunes'와 유사한 느낌으로 음악 서비스/보관함의 이미지를 주기위함

• 의미:

  • EJ: 아내(의주:Eui-Ju)의 이니셜.

  • EJT: Enjoy Together (함께 즐기다).

  • 슬로건 확장: Enjoy Jamming Together (다 같이 음악을 즐기며 어울리자).

2. 핵심 컨셉 (Core Concept)
• 유형: 공간 기반 소셜 주크박스 (Space-based Collaborative Playlist).

• 작동 방식:

  • 호스트(Host): 스피커가 연결된 기기에서 방을 생성하고 음악을 재생.
  • 게스트(Guest): QR코드나 방 코드로 접속하여 플레이리스트 확인 및 곡 추가.

• 음원 소스: 유튜브 검색 API (YouTube Search API) 활용.

3. 주요 기능 및 제어권 (Features & Control)
• 호스트 제어: 원격 제어 가능 (순서 변경, 곡 삭제, 볼륨 조절, 재생/일시정지).

• 권한 공유: 호스트 설정 시 참여한 모든 사용자에게 제어 권한 부여 가능 (민주적 제어).

4. 타겟 및 사용 시나리오 (Target & Scenarios)
• 타겟: 특정 공간에서 음악을 함께 듣는 그룹.

• 사용 사례:

  1. 회사: 팀원들이 함께 노동요(업무용 음악) 리스트 공유.

  2. 카페: 손님이나 직원들이 매장 분위기에 맞는 곡 신청.

  3. 헬스장: 운동할 때 함께 듣는 파이팅 넘치는 음악 공유.
